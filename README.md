# Operaciones_matematicas

Una calculadora básica y modular desarrollada en Python que permite realizar operaciones aritméticas fundamentales mediante la entrada de datos por consola. El proyecto está estructurado en módulos para separar la lógica de las operaciones de la interfaz de usuario.

🚀 Características
Operaciones Básicas: Soporte para suma, resta, multiplicación y división.

Gestión de Errores: Validación integrada para evitar la división por cero.

Modularidad: Las funciones matemáticas están aisladas en un módulo independiente (Operaciones.py), facilitando su reutilización.

🛠️ Tecnologías
Lenguaje: Python 3.x

Paradigma: Programación Modular

📋 Requisitos
Tener instalado Python (versión 3.6 o superior recomendada).

🔧 Instalación y Uso
Descarga los archivos: Asegúrate de tener Operaciones.py y el script principal en la misma carpeta.

Ejecución: Abre una terminal en el directorio del proyecto y ejecuta:

Bash
python principal.py
(Nota: Reemplaza principal.py por el nombre que le hayas dado a tu archivo de ejecución).

Interacción: * Ingresa los dos números cuando se te solicite.

Escribe la operación deseada (suma, resta, multiplicacion, division).

📂 Estructura del Código
Operaciones.py: Contiene la lógica matemática.

sumar(a, b)

restar(a, b)

multiplicar(a, b)

dividir(a, b)
